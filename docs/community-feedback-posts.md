## Reddit draft

I'm either deep in AI psychosis or this is turning into a cool project.

I've been building `Alcove`, a browser-first workspace for steering Codex against real local repos.

The core idea is:

- one durable chat per workspace
- the live artifact stays visible on the right
- run output, checks, and changed files are explicit instead of hidden in a vague agent loop
- studio flows can preview and publish what they're making
- there's also a phone companion so you can check or continue the workspace from your phone

The parts that feel most real right now are:

- Web Studio: chat on the left, live site preview on the right
- Game Studio: the same shell, but for playable artifacts
- `Clear Chat`: reset context without deleting the thing you're building

Repo: `https://github.com/skysmith/alcove-studio`

What I'm trying to figure out before I disappear too far into my own taste:

- does this feel genuinely useful, or just like AI devtool fanfic?
- which part lands immediately?
- what feels overbuilt or confusing?
- if you were going to use it, what would the first real workflow be?

Brutal feedback is welcome.
